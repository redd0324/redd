# The Next Top Professor

一個可在課堂、工作坊直接運行的本地網頁桌遊原型。

## 啟動方式

直接開啟 `index.html`，或使用：

```bash
python3 -m http.server 8000
```

再瀏覽 `http://localhost:8000`。

## 特色

- 4-6 人、4-6 回合可配置
- 9 大價值排序（每回合選前 3）
- 任務卡、事件卡、角色能力
- 即時排行榜與回合歷史紀錄
- 玻璃擬態＋動態光暈動畫風格
